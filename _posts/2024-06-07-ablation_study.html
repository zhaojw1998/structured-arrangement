Ablation Study on Style Prior Architecture

The style prior, inferring multi-track orchestral functions from an input piano score, is a key component of our system. To validate our prior model design, we test with two alternative latent prior architectures based on parallel and delayed code interleaving. We also test with random retrieval of style templates. In our experiment, each architecture is prompted with the same two-bar template and develops the rest of the piece. The generation results are presented with five tabs as follows: 1) Ours, 2) Parallel, 3) Delay, 3) Random, and 3) the input piano score.
